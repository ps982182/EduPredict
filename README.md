# 🚀 EduPredict-X+

AI-powered Academic Risk Intelligence System

## 🌍 Live Demo
👉 https://edu-predict-jet.vercel.app

---

## 🧠 Overview

EduPredict-X+ is a full-stack AI system designed to identify at-risk students and provide intelligent intervention strategies using Machine Learning + Generative AI.

It combines predictive analytics with a natural language AI assistant to help educators make data-driven decisions.

---

## ✨ Features

- 🤖 AI Assistant (LLM-powered using Groq API)
- 📊 Student Risk Prediction Engine
- 📈 Interactive Dashboard & Analytics
- 🧠 Real-time Intervention Suggestions
- ⚡ Fast & scalable full-stack architecture

---

## 🛠️ Tech Stack

**Frontend:** React.js  
**Backend:** Node.js + Express  
**AI/LLM:** Groq (LLaMA 3.1)  
**Deployment:** Vercel (Frontend) + Render (Backend)

---

## ⚙️ How It Works

1. Student data is analyzed using rule-based ML logic  
2. Risk level is predicted (Low / Medium / High)  
3. AI Assistant generates personalized intervention strategies  
4. Insights are visualized via dashboard  

---

## 🚀 Deployment

- Frontend → Vercel  
- Backend → Render  
- API Integration → Groq LLM API
